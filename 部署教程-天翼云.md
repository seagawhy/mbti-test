## 第三步：创建存储桶（Bucket）

### 操作步骤：

1. 在 OOS 控制台，点击 **「创建 Bucket」**

   📸 **截图位置：创建Bucket按钮**
   ```
   [截图说明：Bucket列表页面，蓝色"创建Bucket"按钮]
   ```

2. 填写 Bucket 信息：

   | 配置项 | 建议填写 | 说明 |
   |--------|----------|------|
   | **Bucket名称** | `mbti-test-yourname` | 全局唯一，小写字母、数字、中划线 |
   | **区域** | 选择离你最近的 | 如"华南-广州"、"华东-上海" |
   | **存储类型** | 标准存储 | 静态网站选这个 |
   | **读写权限** | **公共读** ⚠️ | 必须选这个，否则网站无法访问 |

   📸 **截图位置：Bucket配置表单**
   ```
   [截图说明：创建Bucket的表单页面，各项配置已填写]
   
   标注重点：
   - Bucket名称输入框
   - 区域选择下拉框
   - 读写权限选择"公共读"
   ```

3. 点击 **「立即创建」**

4. 创建成功后，在 Bucket 列表中能看到新建的存储桶

   📸 **截图位置：Bucket列表**
   ```
   [截图说明：显示所有Bucket的列表，新创建的Bucket高亮显示]
   ```

## 第七步（可选）：绑定自定义域名

如果你想用自己的域名（如 `mbti.yourdomain.com`）：

### 操作步骤：

1. 在 Bucket 的 **「域名管理」** 中，点击「绑定域名」

   📸 **截图位置：域名管理**
   ```
   [截图说明：域名管理页面，"绑定域名"按钮]
   ```

2. 输入你的域名，如 `mbti.yourdomain.com`

3. 按照提示，在你的域名服务商处添加 CNAME 记录：
   
   | 记录类型 | 主机记录 | 记录值 |
   |----------|----------|--------|
   | CNAME | mbti | xxx.ctyunos.cn |

   📸 **截图位置：CNAME配置示例**
   ```
   [截图说明：显示需要配置的DNS记录信息]
   ```

4. 等待DNS生效（通常10分钟-24小时）

5. 配置 HTTPS 证书（推荐）：
   - 天翼云支持自动申请免费SSL证书
   - 开启 HTTPS 访问更安全

## 🔧 常见问题

### Q1: 网站打不开，显示403错误？
**解决**：检查Bucket的读写权限是否为「公共读」

### Q2: 访问域名显示XML错误？
**解决**：确认已开启「静态网站托管」功能，且默认首页填写正确

### Q3: 如何更新网站内容？
**解决**：直接上传新的 `index.html` 文件，覆盖旧文件即可

### Q4: 可以上传多个文件吗？
**解决**：可以！如果有CSS/JS/图片等资源，一并上传到Bucket根目录
